GMC-16 CPU Architecture Manual
Game Machine CPU — 16-bit Architecture

Specification Version: 1.0
Emulator Version: v4

1. Overview

The GMC-16 is a 16-bit game-oriented CPU architecture designed for a fantasy console environment. It integrates:

A 16-bit general-purpose CPU

A sprite and tile GPU

Hardware collision detection

Memory-mapped IO

Bankswitched cartridge ROM

Framebuffer rendering

Assembler with expression evaluation

The system targets retro-style game development, similar to classic consoles like NES/SNES but with a simpler architecture.

Key features:

Feature	Value
CPU Width	16-bit
Registers	8 general purpose
Address space	64 KB
RAM	8 KB
ROM window	57 KB
Cartridge size	>1 MB
Graphics	256×128 RGB565
Sprites	256
Sprite size	16×16
Tile size	8×8
Framebuffer	Double buffered
Frame rate	60 Hz

        8 KB

        ~48 KB
0xFEFF ─────────────────

0xFF00 ───────────────── IO Registers
        256 bytes
0xFFFF ─────────────────
3.1 RAM
0x0000 – 0x1FFF

Size:

8 KB

Used for:

stack

variables

runtime data

3.2 ROM

ROM contains the game cartridge program.

Two sections exist.

Fixed ROM
0x2000 – 0x3FFF

Always mapped to bank 0.

Size:

8 KB

Usually contains:

boot code

engine

interrupt handlers

Banked ROM
0x4000 – 0xFEFF

Size:

49,152 bytes

This region can switch between 22 banks.

4. Bankswitching System
